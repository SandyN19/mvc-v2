## Starta

Efter att du har säkerhetställt att du har ned alla krav på din enhet

Klona repot med
```
git clone git@github.com:SandyN19/mvc-v2.git
```
Starta serven med:
```
# Alternativ 1
php -S localhost:8888 -t public

# Alternativ 2
symfony server:start
```
